## 项目介绍

本项目由nodejs开发，系统会每日进行预约葫芦娃，i茅台

葫芦娃中奖通知暂时没做，有查询接口的可以在issues中 提供下方便后期对接

在线预约平台正在开发新版本中，原有平台已废弃，请大家敬请期待
